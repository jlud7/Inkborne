# XTeink X4 firmware backups

This folder holds snapshots of the XTeink X4 firmware that's been flashed to the device — both the compiled binary and the game source code that belongs to the custom work layered on top of the upstream `crosspoint-reader` firmware.

## What's here

Each snapshot is a dated folder. A snapshot contains:

- `firmware.bin` — the compiled ESP32-C3 image that was actually flashed to the device
- `SHA256SUMS` — checksums for everything in the snapshot
- `activities-game/` — source snapshot of `src/activities/game/` (the full games suite: Diptych, Caro, Chess, DeepMines, GameOfLife, Maze, Minesweeper, Snake, Solitaire, Sudoku, Tetris, 2048, plus the game picker)
- `game/` — source snapshot of `src/game/` (shared game infrastructure: dungeon generator, renderer, save, state)
- `archive-README.md` and `tracked-ui-diff.patch` — if the snapshot was derived from an earlier local archive, the original README and any UI/theme diffs are preserved here for context
- `README.md` — snapshot-specific notes (what was built, when, against which upstream commit)

## Upstream

The firmware itself is a local fork of `https://github.com/crosspoint-reader/crosspoint-reader`. The custom Games suite and Diptych are not yet pushed to any remote — these snapshots are the only backup of that work until a proper fork is set up.

## What's *not* here

- `firmware.elf` (debug symbols, ~43 MB) and `firmware.map` (linker map, ~15 MB) are intentionally excluded because of size. When a snapshot is made they live alongside the build at `~/Downloads/crosspoint-reader/.pio/build/default/` and can be regenerated by rebuilding the matching source.
- Non-game firmware source (EpubReader, themes, HomeActivity, ActivityManager, etc.). The custom edits to those live in the working checkout at `/Users/james/Downloads/crosspoint-reader` and are not copied into these snapshots. If you want a full firmware-tree backup, that's a larger operation than this folder is scoped for.

## Build / flash workflow

```bash
cd /Users/james/Downloads/crosspoint-reader
pio run -e default                                                    # build
pio run -e default -t upload --upload-port /dev/cu.usbmodem1101       # flash
```

## Snapshots

- `2026-04-12-baseline/` — first backup. Holds the firmware image that was originally built on 2026-04-10. Chapter 1 only, ~1061 lines of `DiptychActivity` code.
- `2026-04-13-full-port/` — first full-parity port of Diptych. Three chapters, 15 shards, bounded 9×9 world, ghouls, pressure plates, mirror mode, hearts, 8 easter eggs, Bloodborne-voice dialogue. Grew `DiptychActivity` to ~2658 lines. This is what's currently running on the device.
